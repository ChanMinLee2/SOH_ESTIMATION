# HI 이론적 분석: LFP (A123) 기준

## 배경

**A123 LFP (LiFePO₄) 셀 열화 메커니즘 우선순위**

| 메커니즘 | 약어 | 진행 속도 | 주요 영향 |
|---------|------|---------|---------|
| 리튬 재고 감소 | LLI | 빠름 (지배적) | 가용 용량 감소, 전극 전위 슬라이딩 |
| 내부 저항 증가 | R↑ | 중간 | 분극 증가, 플래토 기울기 증가, CC→CV 전환 조기화 |
| 활물질 손실 | LAM | 느림 (초~중기 미미) | ICA 피크 감소, 플래토 단축 |

**LFP 충전/방전별 전압 거동 원칙 (예측 방향 결정의 핵심)**

| 구분 | 열화 시 전압 변화 | 이유 |
|------|-----------------|------|
| 방전 | V↓ (하강) | OCV − IR; R↑ → 방전 전압 낮아짐 |
| 충전 | V↑ (상승) | OCV + IR; R↑ → 충전 전압 높아짐 |

→ **방전 구간 전압 통계는 capacity와 양(+) 상관, 충전 구간은 음(−) 상관이 기본값**

| 기호 | 의미 |
|------|------|
| `+●` | 양(+) 상관 / 신호 강 (\|ρ\| > 0.7 예상) |

| `+○` | 양(+) 상관 / 신호 약 (\|ρ\| < 0.4) |

| `−○` | 음(−) 상관 / 신호 약 |
