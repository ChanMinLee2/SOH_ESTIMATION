# Health Indicator (HI) 설명 및 이론적 상관관계

LFP 배터리의 방전 용량(capacity\_Ah)과 각 HI의 예상 Spearman 상관관계를 정리한 문서.

> **LFP 열화 메커니즘 요약 (배경 지식)**
>
> | 메커니즘 | 약어 | 영향 |
> |---------|------|------|
> | 리튬 재고 감소 | LLI | 가용 용량 감소, 전극 전위 이동 |
> | 양극 활물질 손실 | LAM\_pe | 용량 감소, ICA 피크 위치 이동 |
> | 음극 활물질 손실 | LAM\_ne | 용량 감소, ICA 피크 위치 이동 |
> | 내부 저항 증가 | R↑ | 분극 증가, 전압 강하 심화 |
>
> LFP의 특징: 방전 전압 플래토가 매우 평탄(~3.2–3.4V)하여 전압 통계만으로는  
> 열화 신호가 약하게 잡힘. 플래토 **형태** 변화를 포착하는 HI가 유리.

---

## 1. 전압 통계 (Voltage Statistics)

단순 통계량이라 LFP 플래토 평탄성으로 인해 상관이 약한 편.

### v\_mean — 방전 평균 전압

| 항목 | 내용 |
|------|------|
| 계산 | `mean(V)` during discharge |
| 물리적 의미 | 방전 구간 전압의 산술 평균 |
| 열화 연결 | 열화 초기에는 LFP 플래토가 유지돼 거의 변하지 않음. 심화 열화 시 플래토 전압 자체가 낮아지며 감소 |
| 예상 상관 | **양(+), 약~중** — 열화 말기에야 뚜렷해짐 |
| 한계 | LFP 플래토 특성상 초기~중기 열화 구간에서 변별력 낮음 |

---

### v\_std — 방전 전압 표준편차

| 항목 | 내용 |
|------|------|
| 계산 | `std(V)` during discharge |
| 물리적 의미 | 방전 전압 분포의 퍼짐 정도 |
| 열화 연결 | 건강한 LFP는 플래토가 평탄 → v\_std 작음. 열화 시 플래토가 기울어지면 분포가 넓어짐 |
| 예상 상관 | **음(−), 불확실** — 열화 시 v\_std 증가 가능(상관 역전). 데이터셋마다 방향이 달라질 수 있음 |
| 한계 | 방전 전류 크기나 온도 변화에도 영향받아 노이즈 많음 |

---

### v\_skew — 방전 전압 왜도 (Skewness)

| 항목 | 내용 |
|------|------|
| 계산 | `skewness(V)` during discharge |
| 물리적 의미 | 전압 분포의 비대칭성 (양수=오른쪽 꼬리) |
| 열화 연결 | 플래토가 한쪽으로 치우칠 때 변화. 방전 초반 고압 구간 비중이 줄면 분포가 왼쪽으로 이동 |
| 예상 상관 | **불확실** — 전극 열화 방향에 따라 부호가 바뀔 수 있음 |

---

### v\_kurt — 방전 전압 첨도 (Kurtosis)

| 항목 | 내용 |
|------|------|
| 계산 | `kurtosis(V)` during discharge |
| 물리적 의미 | 전압 분포가 얼마나 뾰족한지 (정규분포=0) |
| 열화 연결 | LFP 플래토가 매우 좁은 전압 범위에 집중 → 첨도 큼. 플래토가 무너지면 분포가 넓어져 첨도 감소 |
| 예상 상관 | **양(+), 약** — 플래토 유지 여부와 연동되나 노이즈 큼 |

---

### v\_end — 방전 종료 전압

| 항목 | 내용 |
|------|------|
| 계산 | `V[-1]` (마지막 방전 전압 포인트) |
| 물리적 의미 | 방전이 끝나는 시점의 전압 |
| 열화 연결 | 내부 저항 증가 시 전압 강하가 빨라져 컷오프에 더 일찍 도달 → 종료 전압은 컷오프에 수렴. 반대로 플래토 붕괴 시 저전압 구간이 짧아져 v\_end 변동 |
| 예상 상관 | **불확실** — 데이터 정의(컷오프 조건)에 크게 의존 |

---

### v\_drop — 초반 전압 강하

| 항목 | 내용 |
|------|------|
| 계산 | `V[0] − mean(V[:5%])` (첫 포인트 − 초반 5% 평균) |
| 물리적 의미 | 방전 시작 직후 전압 강하 속도 |
| 열화 연결 | 내부 저항(R) 증가 → 분극 심화 → 초반 강하 증가. `ΔV ≈ I × R` 관계 |
| 예상 상관 | **음(−)** — v\_drop 증가 시 열화 진행 = 용량 감소 |
| 주의 | 방전 전류가 일정하지 않으면 I×R drop 해석이 달라짐 |

## 2. 에너지 기반 (Energy-based)

### energy\_Wh — 방전 에너지

| 항목 | 내용 |
|------|------|
| 계산 | `∫V · |I| · dt / 3600`  [Wh] |
| 물리적 의미 | 해당 사이클에서 실제로 방출된 전기 에너지 |
| 열화 연결 | 에너지 = 용량 × 평균 전압. 열화 시 용량 감소 + 전압 저하 → 에너지 감소가 용량 감소보다 가파름 |
| 예상 상관 | **강한 양(+)** — capacity와 거의 선형. 가장 직접적인 열화 지표 중 하나 |
| 활용 | SOH 추정 레이블 대안으로도 사용 가능 |

---

### v\_energy — 에너지 가중 평균 전압

| 항목 | 내용 |
|------|------|
| 계산 | `energy_Wh / capacity_Ah`  [V] |
| 물리적 의미 | 단위 용량당 에너지, 즉 용량과 독립적인 전압 프로파일 변화 측정 |
| 열화 연결 | 분극 증가 → 실효 전압 감소 → v\_energy 감소. capacity와 독립적으로 전압 열화를 포착 |
| 예상 상관 | **양(+)** — 플래토 전압 저하, 분극 증가와 연동 |
| v\_mean 대비 장점 | 시간 평균이 아닌 용량 가중 평균이라 실제 전기화학적 의미에 더 부합 |

## 3. SOC별 전압 프로파일 (Voltage at specific SOC)

누적 방전 전류 적산으로 SOC 대응 위치 추정.  
`q_cum = ∫|I|·dt`, 특정 분율(20/50/80%)에서의 전압을 추출.

### v\_at\_q20 — Q=20% 시점 전압

| 항목 | 내용 |
|------|------|
| 물리적 의미 | 방전 초반 (전체 용량의 20% 소진 시점) 전압 |
| 열화 연결 | LFP 방전 초반: 고압 구간(~3.4V)에서 플래토로 전환되는 구간. 내부 저항 증가 → 이 구간에서 더 일찍 전압 강하 |
| 예상 상관 | **양(+)** — 열화 시 v\_at\_q20 감소 경향 |

---

### v\_at\_q50 — Q=50% 시점 전압

| 항목 | 내용 |
|------|------|
| 물리적 의미 | 방전 중간 (플래토 중심부) 전압 |
| 열화 연결 | LFP 플래토 심부는 매우 안정적이라 변화가 작음. LLI 심화 시 전극 전위 이동으로 미세하게 이동 가능 |
| 예상 상관 | **양(+), 약** — 플래토 특성상 신호 작을 것으로 예상 |
| 참고 | 신호가 약하면 모델에서 weight가 낮게 설정될 수 있음 |

---

### v\_at\_q80 — Q=80% 시점 전압

| 항목 | 내용 |
|------|------|
| 물리적 의미 | 방전 후반 (전체 용량의 80% 소진 시점, 플래토 이탈 직전~후) 전압 |
| 열화 연결 | 플래토 이탈 후 급격한 전압 강하 구간. 열화 시 이 구간이 더 일찍, 더 빠르게 나타남 → v\_at\_q80 낮아짐 |
| 예상 상관 | **양(+), 강** — SOC 기반 HI 중 가장 민감할 것으로 예상 |
| 주의 | 방전 종료 조건에 따라 Q=80%가 실제 데이터에 존재하지 않는 셀이 있을 수 있음(NaN 처리) |

## 4. 부분 전류 적산 (Voltage-window Partial Integration)

`Q_window = ∫|I|·dt` 를 전압 구간별로 적산.  
LFP의 전압 구간 특성을 직접 포착.

### q\_high\_v — V > 3.2V 구간 용량

| 항목 | 내용 |
|------|------|
| 계산 | `∫|I|·dt / 3600` where `V > 3.2V`  [Ah] |
| 물리적 의미 | 플래토 + 플래토 상단 구간에서 방출된 용량 |
| 열화 연결 | LFP 플래토(3.2~3.45V)가 대부분의 용량을 담당. 열화 시 플래토 단축 → q\_high\_v 감소 |
| 예상 상관 | **강한 양(+)** — 총 용량의 변화와 거의 동행 |

---

### q\_tail — V < 3.2V 구간 용량

| 항목 | 내용 |
|------|------|
| 계산 | `∫|I|·dt / 3600` where `V < 3.2V`  [Ah] |
| 물리적 의미 | 방전 말기 저전압 구간(꼬리 부분) 용량 |
| 열화 연결 | 두 가지 경쟁 효과가 있음:<br>① 전체 용량 감소 → tail도 감소 (양의 상관 방향)<br>② 내부 저항 증가 → 전압 강하 가속 → tail 비중 증가 (음의 상관 방향)<br>어느 효과가 우세한지 데이터 확인 필요 |
| 예상 상관 | **불확실 (양 또는 음)** — q\_plateau\_ratio와 함께 해석해야 의미 명확 |

---

### q\_plateau\_ratio — 플래토 구간 용량 비율

| 항목 | 내용 |
|------|------|
| 계산 | `Q[3.2V≤V≤3.45V] / Q_total` |
| 물리적 의미 | 전체 방전 용량 중 LFP 주 플래토 구간이 차지하는 비율 |
| 열화 연결 | LLI 주도 열화: 용량 전체가 비례 감소 → 비율 유지.<br>저항 증가: 플래토 전압이 3.2V 아래로 내려가는 비중 증가 → 비율 감소.<br>LAM: 플래토 자체가 짧아지며 비율 감소 |
| 예상 상관 | **양(+), 중~강** — 저항 증가/LAM 열화 시 유효. LLI 단독 시 신호 약함 |
| 활용 | q\_tail과 함께 쓰면 열화 메커니즘 식별에 활용 가능 |

## 5. 방전 시간 (Discharge Time)

### t\_discharge — 방전 소요 시간

| 항목 | 내용 |
|------|------|
| 계산 | `t[-1] − t[0]`  [s] |
| 물리적 의미 | 한 방전 사이클의 총 소요 시간 |
| 열화 연결 | 정전류(CC) 방전 시 `t = Q / I` → 용량에 선형 비례 |
| 예상 상관 | **강한 양(+)** — 거의 capacity의 선형 변환 |
| 한계 | 전류가 일정하면 t\_discharge는 capacity와 중복되는 정보가 많아 독립적 기여가 적음. 다단계 방전(HUST의 D0~D3)에서는 각 단계 시간이 더 유용 |

## 6. ICA — 증분 용량 분석 (Incremental Capacity Analysis, dQ/dV)

전압 구간별 용량 증분 `dQ/dV` 를 계산.  
LFP의 평탄한 플래토는 ICA에서 **뾰족하고 높은 피크**로 나타남.  
피크의 형태 변화가 곧 플래토(전극 상전이) 건전성 변화.

```
     dQ/dV
      │         ↑ 건강한 셀 (높고 뾰족)
      │        /|\
      │       / | \
      │    ─ /  |  \ ─  열화 셀 (낮고 넓음)
      │───────────────────→ Voltage [V]
           3.2  3.35 3.5
```

### ica\_peak\_h — ICA 피크 높이

| 항목 | 내용 |
|------|------|
| 계산 | `max(dQ/dV)` in 3.2~3.5V after Savitzky-Golay smoothing |
| 물리적 의미 | LFP 상전이 피크의 선명도. 피크가 클수록 플래토가 뚜렷하고 상전이가 가역적 |
| 열화 연결 | LAM·LLI 진행 → 상전이 가역성 저하 → 피크 감소 및 넓어짐 |
| 예상 상관 | **양(+), 강** — 열화 핵심 지표 중 하나 |

---

### ica\_peak\_v — ICA 피크 전압

| 항목 | 내용 |
|------|------|
| 계산 | `argmax(dQ/dV)` 의 전압 위치 |
| 물리적 의미 | LFP 상전이가 발생하는 전압 위치 |
| 열화 연결 | LLI: 음극·양극 전위 이동(슬라이딩) → 피크 위치 이동. 양극 LAM: 피크 전압 이동 방향이 달라짐 |
| 예상 상관 | **불확실 (열화 메커니즘에 의존)** — 방향은 데이터로 확인 필요. 다른 HI와 함께 사용 시 메커니즘 식별 보조 가능 |

---

### ica\_peak\_area — ICA 피크 면적

| 항목 | 내용 |
|------|------|
| 계산 | `∫max(dQ/dV, 0) dV` in 3.2~3.5V |
| 물리적 의미 | 플래토 구간에서 방출된 총 용량의 근사. 면적 = `∫dQ/dV·dV = ΔQ` |
| 열화 연결 | 플래토 용량이 줄면 면적도 줄어듦 |
| 예상 상관 | **강한 양(+)** — ica\_peak\_h보다 노이즈 강인함. 피크가 넓어지더라도 면적은 실제 플래토 용량 반영 |
| 권장 | ica\_peak\_h와 함께 쓰면 "피크가 작아졌는지" vs "피크가 넓어졌는지" 구분 가능 |

## 7. DVA — 미분 전압 분석 (Differential Voltage Analysis, dV/dQ)

ICA의 역수 관계: `dV/dQ = 1 / (dQ/dV)`.  
Q(누적 용량)를 x축으로 전압 프로파일을 미분.

```
     dV/dQ
      0 │──────────────────────────── 건강한 셀 (플래토에서 ≈ 0)
        │
     −  │  \   /
        │   \_/  열화 셀 (플래토에서 더 깊은 골짜기)
        │──────────────────────────→ Q [Ah]
           플래토 구간
```

### dvdq\_min — 플래토 구간 dV/dQ 최솟값

| 항목 | 내용 |
|------|------|
| 계산 | `min(dV/dQ)` 를 전압 3.15~3.50V에 해당하는 구간에서 추출 |
| 물리적 의미 | 플래토 구간에서 단위 용량당 전압이 얼마나 빠르게 변하는지 |
| 열화 연결 | 건강한 LFP: 플래토에서 dV/dQ ≈ 0 (전압 거의 불변)<br>열화 시: 플래토 기울기 증가 → dV/dQ가 더 음수가 됨<br>저항 증가: 분극 → 플래토 전압 하락 속도 증가 |
| 예상 상관 | **양(+)** — dvdq\_min이 0에 가까울수록 건강함, 더 음수일수록 열화 진행 |
| ICA 대비 장점 | 낮은 SNR에서도 골짜기 구조는 비교적 안정적 |

## 8. 쿨롱 효율 (Coulombic Efficiency)

### ce — 쿨롱 효율

| 항목 | 내용 |
|------|------|
| 계산 | `Q_discharge / Q_charge` (같은 사이클의 방전/충전 용량 비율) |
| 물리적 의미 | 충전 중 입력된 전하 대비 방전으로 회수된 전하의 비율 |
| 열화 연결 | 부반응(SEI 성장, 리튬 석출, 전해질 분해) 발생 시 충전 중 일부 전하가 소모 → CE 감소.<br>장기적으로 CE 저하가 누적되면 용량 감소 |
| 예상 상관 | **양(+), 이론적** — CE가 낮을수록 용량 손실이 빠름 |
| 주의 및 한계 | LFP는 CE가 이미 매우 높아(0.999 이상) 변화 폭이 수백 mAh급 측정 정밀도 이하일 수 있음. 사이클 간 노이즈가 신호보다 클 수 있어 **장기 추세(rolling average)** 로 보는 것이 바람직 |

## 활용 지침

1. **강한 양(+) 상관 HI** (`energy_Wh`, `q_high_v`, `ica_peak_area`, `t_discharge`)는 모델 feature로서 직접 활용도 높음. 단, `t_discharge`는 capacity와 거의 선형이라 단독 사용 시 trivial할 수 있음.

2. **플래토 형태 HI** (`ica_peak_h`, `dvdq_min`, `q_plateau_ratio`)는 단독 상관은 중간이나 **열화 메커니즘 구분**에 중요. LLI vs LAM vs R 증가를 구분하는 데 조합 사용 권장.

3. **LFP 특이 주의사항**: `v_at_q50`은 플래토 깊은 곳이라 신호가 거의 없을 수 있음. 반면 `v_at_q80`은 플래토 이탈 구간이라 민감.

4. **ce (쿨롱 효율)**은 이론적으로 중요하나 측정 노이즈가 신호를 압도할 수 있어 단독 사용보다 **10~50 사이클 이동 평균** 후 사용 권장.

5. **데이터셋 간 방향 불일치**: MIT와 HUST의 전류 부호 관례가 반대(`current_A`)이나 `i_mag = |I|` 로 처리하여 통일. 단, 일부 HI(특히 phase 경계 기반)는 데이터셋 간 상관 부호가 다를 수 있어 개별 확인 필요.

6. **세그먼트 ICA/DVA 수치 불안정 주의**: 플래토 세그먼트(`s_mid`, `chg_s_mid`)는 전압 범위가 수십 mV 이하로 좁아 `ica_peak_h`, `dvdq_min`이 극단값을 가질 수 있음. `chg_s_hi` 구간은 CV 충전 포함으로 `dV ≈ 0 → dQ/dV → ∞` 발생. 해당 HI 사용 시 물리 범위 클리핑 또는 유효성 검사 필요.

7. **HUST 비정상 종료 사이클**: `v_end < 1.9V` 사이클 2.88%(4,214건)가 존재하며 `v_end`, `v_drop`, `energy_Wh` HI를 왜곡함. `preprocess.py` 실행으로 제거 가능 (→ `dataset_anomalies.md` 참조).

## SoC 구간 분할 설계 근거

### 전압 기준 vs 누적 전하량 비율(q_frac) 기준

HI 추출 시 사이클을 SoC 구간별로 나눌 때 **전압 임계값**을 경계로 삼는 방법과  
**누적 전하량 비율(q_frac)** 을 경계로 삼는 방법 중 후자를 채택했다.

```
q_frac = cumsum(|I| · dt) / Q_total   (0 ~ 1)
```

#### LFP 플래토 문제

LFP 양극 특성상 3.2 ~ 3.45 V 구간(플래토)에 전체 용량의 70 ~ 80 %가 집중된다.  
전압 임계값으로 구간을 나누면:

| 전압 구간 | 실제 포함 용량 | 문제 |
|-----------|---------------|------|
| V < 3.2 V | ~15 % | 데이터 극소수 |
| 3.2 ~ 3.4 V | ~70 % | 모든 데이터 쏠림 |
| V > 3.4 V | ~15 % | 데이터 극소수 |

플래토에 데이터가 몰려 구간 분할이 사실상 의미 없어진다.

#### q_frac 기준의 장점

| 항목 | 전압 기준 | q_frac 기준 |
|------|-----------|-------------|
| 구간 내 데이터 수 | 불균등 (플래토 쏠림) | 균등 (항상 전체의 일정 비율) |
| 열화 영향 | 전압 곡선 이동 → 경계 틀어짐 | 총 Q 대비 비율이므로 열화와 무관 |
| C-rate 영향 | IR drop으로 전압 왜곡 | 상대적으로 작음 |
| SoC 근사 정확도 | LFP 비선형 OCV → 부정확 | Q ∝ SoC이므로 선형 근사 |

#### q_frac 기준의 한계

- **절대 SoC 이동**: 열화가 진행될수록 Q_total이 감소하므로 q_frac 40 %의 절대 Ah가 달라진다. 사이클 초반과 말기에 동일한 q_frac 구간이 약간 다른 SoC 위치를 가리킬 수 있다.
- **부분 충방전 민감**: Q_total이 작은 부분 사이클이 섞이면 q_frac 의미가 희석된다.

#### 구간 경계 설정 (0.4 / 0.7)

SoC 경계를 0.3 / 0.6 대신 q_frac 0.4 / 0.7 로 설정한 이유:  
LFP 플래토(SoC 약 20 ~ 80 %)가 실제 용량 기준으로 넓게 걸쳐 있어  
중간 구간이 플래토 핵심을 충분히 포함하도록 경계를 약간 바깥으로 잡았다.

| 세그먼트 | q_frac | 근사 SoC | 물리적 의미 |
|----------|--------|----------|-------------|
| `s_hi` (방전) / `chg_s_lo` (충전) | 0 ~ 40 % | 60 ~ 100 % / 0 ~ 30 % | 방전 초반(고전압) / 충전 초반(저전압) |
| `s_mid` / `chg_s_mid` | 40 ~ 70 % | 30 ~ 60 % | 플래토 중심 |
| `s_lo` (방전) / `chg_s_hi` (충전) | 70 ~ 100 % | 0 ~ 30 % / 60 ~ 100 % | 방전 후반(저전압) / 충전 후반(CV 포함) |

방전과 충전은 진행 방향이 반대(SoC 감소 vs 증가)이나 q_frac 계산 로직은 동일하며 SoC 라벨만 반전된다.
